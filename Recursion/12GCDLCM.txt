GCD and LCM

Difficulty: EasyMax Score: 25
25 Points
Problem Description
Write a program to print the Greatest Common Divisor (GCD) and Lowest Common Multiple (LCM) of two numbers.

You are required to calculate and print the GCD of the two numbers. Additionally, you need to compute and print the LCM of the same numbers.

Input Format
The first line contains an integer num1.
The second line contains an integer num2.

Output Format
Print the GCD on the first line and the LCM on the second line, where a is the GCD and b is the LCM.


Example
Example 1

Input

8 

12 

Output

4 

24

Example 2 

Input

18 

30 

Output

6 

90


Problem Constraints
2 < = N < = 10^9
