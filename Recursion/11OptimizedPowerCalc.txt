Optimized power calculation

Difficulty: EasyMax Score: 25
25 Points
Problem Description
You are given two integers, A and B. You need to find B raised to the power A in O(log⁡(A))O(\log(A))O(log(A)) time complexity using recursion.

Note: Complete the given function and use recursion to solve this problem. It is guaranteed that the test cases are constructed such that the result does not overflow a 64-bit integer datatype.

Input Format

The first line contains two integers, A and B.

Output Format

Print the result.


Example
Example 1

Input
2 3

Output
8

Explanation
2 raised to the power 3 is 8


Example 2

Input
10 5

Output
100000

Explanation
10 raised to the power 5 is 100000


Problem Constraints
1 <= A <= 64
1 <= B <= 10^8


Topics
Recursion
