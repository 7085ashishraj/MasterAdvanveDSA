Recursively Print Numbers

Difficulty: EasyMax Score: 25
25 Points
Problem Description
Write a recursive program to print all integers from 1 up to M. The sequence should be 1 2 3 ... M.

Input Format

Input consists of a single integer M.

Output Format

Print the numbers from 1 to M in ascending order, each separated by a single space.


Example
Example 1

Input
4

Output
1 2 3 4

Explanation
The numbers from 1 to 4 are printed in ascending order.


Example 2

Input
7

Output
1 2 3 4 5 6 7

Explanation
The numbers from 1 to 7 are printed in ascending order.


Problem Constraints
1 <= M <= 1000
